# OTO MAZLUM

Otomotiv yedek parça web sitesi.

## Vercel'e Deployment

Bu proje Vercel'e kolayca deploy edilebilir:

1. Vercel hesabınızla GitHub reponuzu bağlayın
2. Otomatik deploy başlayacaktır
3. Node.js 18 versiyonu kullanılacaktır

## Yerel Geliştirme

```bash
npm install
npm run dev
```

## Özellikler

- Static HTML website
- Bootstrap ve jQuery kullanımı
- Responsive tasarım
- Owl Carousel entegrasyonu

## Node.js Versiyonu

Bu proje Node.js 18+ gerektirir ve Vercel'de bu versiyon kullanılacak şekilde yapılandırılmıştır.
